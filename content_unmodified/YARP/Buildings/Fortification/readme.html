Description

This is my pack of fortification buildings for my wall system. I made them to have buildable and walkable walls (via triggers) with different animations for each level of destruction (see description of the different models). The wall system also features the abillity to turn the walls before building and has a height adjustment system, raising/lowering the terrain beneath the walls gradually to avoid big height differences between the wall sections.

I also made a presentation map with the wall system. There you can see how the models are intended to work (with some basic gameplay for testing and fun ).

The models only uses ingame textures (for low filesize), have a portrait camera and team colour.
