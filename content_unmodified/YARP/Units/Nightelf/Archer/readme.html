Description

This one was going to be a simple edit of the archer, changing the weird potato sack boots and cleaning the wrapping on the arrow and bow, and it snowballed.

Is based on the same body that the Lynxrider. Bow and arrow were copy/reversed/pasted to the sides with better wrapping, the bowstring was added using the unused string on the ballista skin.

The bundle has a set of arrows, from the basic missile to any upgrade like Fire/Ice/Poison arrows.
