Drealord (Pre-Beta Look-A-Like)

Model by: Dan van Ohllus (aka Kuhneghetz)

Descriptions:
I've got inspired by the Dreadlords from the alpha and pre-beta versions of Warcraft III, so I've made a little remake from it to make this model look similar.
Based on the Dreadlord model with a new walk animation, because skirt looked a bit awkward when the old walk animation was played.
And all textures are ingame, so don't worry about special paths.

No custom icons are needed since the ingame Dreadlord icon fits very well.
